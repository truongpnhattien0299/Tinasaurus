Cập nhật thiết bị

## Request

| Level | Field Name      | Data Type  | Required | Description                                                                    |
| ----- | --------------- | ---------- | -------- | ------------------------------------------------------------------------------ |
| 1     | RequestID       | String(50) | M        | GUID                                                                           |
| 1     | RequestDateTime | String(50) | M        | yyyyMMddHHmmss                                                                 |
| 1     | Function        | String     | M        | Trùng với tên API                                                              |
| 1     | Data            | String     | M        | Chứa [Data Content of request](#data-content-of-request) thay đổi tùy theo API |

## Data content of request

| Level | Field Name  | Data Type | Required | Description                 |
| ----- | ----------- | --------- | -------- | --------------------------- |
| 2     | CIFNumber   | int       | M        | Số CIF của Khách hàng       |
| 2     | Location    | Object    | M        | Thông tin vị trí khách hàng |
| 3     | latitude    |           |          | Kinh độ                     |
| 3     | longtitude  |           |          | Vĩ độ                       |
| 3     | geoLocation |           |          | Vị trí trên bản đồ          |
| 3     | DeviceInfo  | Object    | M        | Thông tin thiết bị          |
| 3     | appCodeName |           |          | Mã code app                 |
| 3     | appName     |           |          | Tên app                     |
| 3     | appVersion  |           |          | Phiên bản app               |
| 3     | language    |           |          | Ngôn ngữ                    |
| 3     | deviceType  |           |          | Loại thiết bị               |
| 3     | deviceId    |           |          | Mã thiết bị                 |
| 3     | deviceName  |           |          | Tên thiết bị                |
